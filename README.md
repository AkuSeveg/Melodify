# 🎧 Melodify

Melodify adalah aplikasi web pencarian musik modern dengan antarmuka premium bergaya *Deep Black & Green* (terinspirasi dari Spotify). Ditenagai oleh **Node.js** di sisi *backend* dan **YouTube Music API**, aplikasi ini memungkinkan pengguna untuk mencari dan memutar lagu dengan sangat cepat tanpa iklan.

## ✨ Fitur Utama
- **Desain Kelas Atas:** UI/UX modern dengan efek kartu melengkung, transisi halus, dan animasi *Skeleton Loading* saat memuat lagu.
- **Pencarian Cepat:** Mengambil data langsung dari basis data YouTube Music resmi.
- **Navigasi SPA:** Berpindah menu (Home & About) secara instan tanpa perlu *reload* halaman.
- **Serverless Ready:** Arsitektur *backend* yang sudah dioptimalkan penuh untuk *deployment* gratis di Vercel tanpa memicu *error*.

## 🛠️ Tech Stack
- **Frontend:** HTML5, Modern CSS (Grid/Flexbox), Vanilla JavaScript (ES6+)
- **Backend:** Node.js, Express.js
- **Scraper API:** `yt-search`
- **Deployment:** Vercel (Serverless Functions)
